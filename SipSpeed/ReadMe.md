# SipSpeed

### 개요

* C++ SIP stack / SipUserAgent 를 이용한 MFC 기반 SIP 성능 테스트 프로그램 개발
* C++ SIP User Agent 를 이용하여서 SIP 서버 성능 측정 프로그램을 개발한다.

### 주요 기능

* CPS 측정
