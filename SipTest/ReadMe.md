# SipTest

### 개요

* C++ SIP stack / SipUserAgent 를 이용한 MFC 기반 SIP 서버 기능 테스트 프로그램 개발
* C++ SIP User Agent 를 이용하여서 SIP 서버 기능 테스트 프로그램을 개발한다.

### 주요 기능

* 로그인 테스트
* 통화 연결 테스트
* 통화 취소 테스트
* 통화 거절 테스트
* 통화 전달 테스트
