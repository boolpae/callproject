# SipUserAgentMFC

### 개요

* C++ SIP stack / SipUserAgent 를 MFC 에서 편리하게 사용하기 위한 MFC 라이브러리 개발
* C++ SIP User Agent 를 이용하여서 MFC 기반 SIP callback 라이브러리를 개발한다.

### 주요 기능

* 로그인 기능
* 통화 시작/취소/종료 기능
